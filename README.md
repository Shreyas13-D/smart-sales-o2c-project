# Smart Sales Analytics & Order-to-Cash (O2C) System

## Overview
This project implements a complete 7-step Order-to-Cash (O2C) workflow integrated with a sales analytics dashboard.

## Features
- KPI Dashboard (Revenue, Orders)
- Revenue Trend Analysis
- Customer Insights
- Product Performance
- O2C Workflow Simulation

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## Dataset
CSV-based structured data including:
order_id, customer_id, order_date, product_sku, total_amount, status fields

## Run Project
streamlit run app.py
